# EMPLOYEE DIRECTORY
### A simple app used to keep track of an employee data base allowing quick view of the team and some employment information 

## Technoligies Used 
- React 
- Random User API
- Bootstrap
- JSX

## Installation 
- npm install
- npm create-react-app
- npm start
- npm install --save bootstrap
